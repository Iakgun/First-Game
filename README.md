  第一个游戏-横版过关
==========

第一个游戏为横版过关游戏，通过8次练习，教会大家如何用Cocos2d-x制作一个完整的横版过关游戏。

`注意`:此文件仅由gloryming修改和更新。

项目安排及说明：本仓库共5名成员，每名成员在根目录下新建一个以自己名字命名的文件夹，文件夹里面存放各自的项目文件。所有项目图片资源都在`src`文件夹里面，每次任务更新一次资源。
